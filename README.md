# 🚀 마크다운 테스트 페이지 (H1)
이 파일은 Markdown 렌더링 스타일과 기능을 테스트하기 위한 샘플 문서입니다.

## 📌 문서 개요 (H2)
마크다운을 사용하면 텍스트 기반으로도 깔끔하게 구조화된 문서를 작성할 수 있습니다. 

### 🔍 주요 기능 확인 (H3)
아래 요소들이 정상적으로 표시되는지 확인해 보세요.

* **굵은 글씨 (Bold)** 및 *기울임꼴 (Italic)* 적용 여부
* ~~취소선 (Strikethrough)~~ 기능
* `인라인 코드(Inline Code)` 스타일
